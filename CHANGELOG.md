### Added
- Initial project structure with SwiftUI MenuBarExtra
- 8 snap actions: left/right half, four quarters, maximize, center
- Global keyboard shortcuts via KeyboardShortcuts package
- AXUIElement-based window management (get/set position and size)
- Multi-monitor screen detection with AX/NS coordinate conversion
- Accessibility permission check and prompt
- Menu bar UI with snap action buttons
- Unit tests for SnapCalculator (10 tests, all passing)
