# Zachs AI Brand Guidelines

Apply these guidelines when styling UI, creating screenshots, or building visual assets for any Zachs AI product.

For full specs, SwiftUI patterns, and logo details, read [references/brand-system.md](references/brand-system.md).

## Quick Reference

### Colors
| Token | Hex | Use |
|-------|-----|-----|
| Accent | `#7BFF00` | Interactive elements, highlights, CTAs |
| Light | `#F1F1F1` | Light backgrounds, secondary text on dark |
| Dark | `#221A10` | Text on light backgrounds |
| Secondary | `#06024C` | Primary dark backgrounds, navy theme |
| Variant | `#0C0472` | Gradient companion to Secondary |

### Typography
- **Main:** Funnel Display (headings, bold UI text)
- **Accent:** Instrument Serif Italic (decorative, contrast)
- **Fallback:** SF Pro (macOS system font) when brand fonts unavailable

### Personality
Playful, accessible, not jargon-heavy. Like an expert colleague — technical depth with friendly communication.
